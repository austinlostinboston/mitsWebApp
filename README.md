## Installation 

### liblinear

> 1. extract liblinear-1.96.zip
> 2. cd liblinear-1.96
> 3. make
> 4. cd python;make
> 5. after step4, a file called liblinear.so.2 will be created under the directory that you just extracted
> 6. copy/replace that file into mitsWebApp/weiss/
